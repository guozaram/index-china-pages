# index-china-pages

本仓库用于归档和发布多个独立 HTML 页面。这些页面彼此独立，不针对任何特定域名或具体网站。

## 项目简介

本项目旨在提供一个集中存放和展示 HTML 页面的空间。所有页面均为独立文件，可直接通过 GitHub Pages 或其他静态托管服务访问。仓库内容不包含动态后端逻辑，仅用于呈现静态网页。

## 目录说明

```
/
├── pages/          # 存放所有独立 HTML 页面
├── assets/         # 公共资源文件（如 CSS、JS、图片等）
├── index.html      # 根目录页面（可选，用于导航或说明）
└── README.md       # 本文件
```

- `pages/`：每个子目录或文件代表一个独立的页面项目。
- `assets/`：存放页面共用的样式、脚本或媒体文件。
- `index.html`：可选的入口页面，便于浏览仓库内容。

## 页面归档说明

- 每个 HTML 页面均为自包含文件，尽量通过相对路径引用资源。
- 页面内容保持原始状态，不进行压缩或混淆。
- 归档的页面可能来自不同来源或不同时期，风格与功能各异。
- 不保证所有页面均能完美兼容所有浏览器或设备。

## 使用方式

1. 克隆或下载本仓库到本地。
2. 直接在浏览器中打开 `pages/` 下的 HTML 文件即可查看。
3. 若需部署到线上，可将整个仓库上传至支持静态页面的托管服务。
